# Ines Duarte — UX/UI Portfolio

A simple, feminine portfolio site for a UX/UI design student. The site highlights an introduction, about section with downloadable CV, experience and skills, project case studies, and contact details.

## Getting started

Open `index.html` in your browser to explore the portfolio. Each project card links to a dedicated case study page.

## Structure

- `index.html` – landing page with hero, about, experience, projects and contact sections.
- `project-aurora.html` – telehealth case study.
- `project-bloom.html` – habit coaching case study.
- `project-storyline.html` – digital library design system case study.
- `styles.css` – global styles using dusty pink tones and soft shadows.
- `assets/images/profile-portrait.svg` – portrait illustration used in the about section.
- `assets/cv/ines-duarte-cv.pdf` – placeholder CV for download links.

Feel free to customise colours, copy, and assets to make it your own.
